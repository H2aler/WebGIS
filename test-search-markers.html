🔍 검색 결과 마커 기능 테스트

📋 테스트 방법

  • 1. 메인 WebGIS 애플리케이션을 열고 검색창에 도시나 나라를 입력하세요
  • 2. 검색 결과 중 하나를 클릭하면 지도에서 해당 위치로 이동하고 마커가 표시됩니다
  • 3. 우측 사이드바의 "🔍 검색 결과 마커" 패널에서 추가된 마커들을 확인할 수 있습니다
  • 4. 각 마커에는 🚀 이동 버튼과 🗑️ 삭제 버튼이 있습니다
  • 5. "🗑️ 모두 지우기" 버튼으로 모든 마커를 한 번에 삭제할 수 있습니다
